# 2026-07-24 개발일지

## 오늘의 요약

어제 제출한 Play Store 비공개 테스트가 카테고리 정책 위반으로 거절됐다. 카테고리를 의료(Medical)에서 생산성/라이프스타일로 변경해 재제출했고, 현재 심사 중이다. 내부 테스트 트랙에는 이미 versionCode 56이 "내부 테스터에게 제공됨" 상태로 활성화되어 있어 바로 테스트 가능한 상태임을 확인했다.

## 세션: Play Store 심사 거절 대응

### 거절 사유

```
Violation of Play Console Requirements
Some types of apps can only be distributed by organizations.
You have selected an app category or declared your app offers
certain features that require you to submit your app using
an organization account.
```

**원인**: 카테고리를 "의료(Medical)"로 설정했기 때문. Google Play는 의료 카테고리 앱을 개인 계정이 아닌 조직(Organization) 계정에서만 배포 가능하도록 제한한다.

### 해결 방법 및 적용

- Play Console → 스토어 설정 → 앱 카테고리를 "의료" → **"생산성(Productivity)"** 또는 **"라이프스타일(Lifestyle)"** 으로 변경
- 변경 후 재제출 → "변경사항을 검토 중입니다" 상태로 전환
- 실버링크는 의료 앱이 아니라 "AI 기반 가족 소통·케어 인사이트" 앱이므로 생산성/라이프스타일이 앱 성격에도 더 정확함

### 잠재적 추가 이슈: READ_CALL_LOG

- Google의 **제한 권한(Restricted Permission)** 그룹에 속하는 권한
- 기본 전화 앱이 아닌 앱이 사용하려면 심사관이 추가로 사용 목적 확인 요청 가능
- 검토 안내에 "AI 통화 분석을 위해 통화 기록을 읽어 어르신 상태를 파악한다"는 내용이 명시되어 있으면 통과 가능성 높음
- 거절 시 추가 설명 이메일이 별도로 옴

### 내부 테스트 현황 확인

- 내부 테스트 트랙에 **versionCode 56 (1.0.55)** 이미 "내부 테스터에게 제공됨" 상태
- 게시일: 2026-07-22, 검토 없이 즉시 활성화
- 비공개 테스트 심사 기다리지 않고 내부 테스트로 엄마 폰에서 바로 테스트 가능
- 테스터 탭에서 Google 계정 이메일 확인 후 옵트인 링크 전송하면 Play Store에서 즉시 설치 가능
- versionCode 58 최신 버전으로 내부 테스트도 원하면 versionCode 59 빌드 후 내부 테스트 트랙에 업로드하면 됨

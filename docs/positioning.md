# SilverLink AI — 포지셔닝 + 경쟁사 분석

> Day 32 산출물. "왜 SilverLink인가"를 내·외부 커뮤니케이션에 일관되게 적용하기 위한 레퍼런스.

---

## 핵심 포지셔닝 (3문장)

1. **이미 하고 있는 행동을 강화합니다.**  
   가족이 부모님께 전화하는 일은 이미 일어나고 있습니다. SilverLink는 그 통화를 녹음 → AI 분석으로 연결해 추가 부담 없이 돌봄 데이터를 만듭니다.

2. **어르신 건강에 특화된 유일한 통화 분석 앱입니다.**  
   클로바 노트·Google Recorder는 회의와 메모를 위해 설계됐습니다. SilverLink는 외로움·인지 저하·낙상 위험 등 노인 돌봄 전문가가 주목하는 7가지 신호를 처음부터 타겟으로 만들었습니다.

3. **멀리 사는 가족도 놓치지 않습니다.**  
   위험 신호가 감지되면 즉시 안전 알림이 도착합니다. 매주 케어 타임라인과 사회 연결 점수로 부모님의 상태 변화를 추적할 수 있습니다.

---

## 타겟 고객

| 세그먼트 | 페인 포인트 | SilverLink 해결책 |
|----------|------------|------------------|
| 30~50대 자녀 (부모 비거주) | "전화하면 괜찮다고 하는데 정말인지 모르겠다" | AI 통화 분석 → 신호 감지 |
| 맞벌이 부부 | "자주 못 찾아뵙는 죄책감" | 통화 한 번으로 돌봄 기록 자동화 |
| 원거리 가족 | "응급 상황을 너무 늦게 알게 될까 봐" | 위험 신호 즉시 알림 |

## 경쟁사 분석

### 1. 네이버 클로바 노트

| 항목 | 내용 |
|------|------|
| 주 목적 | 회의 전사·요약 |
| 강점 | 한국어 인식 정확도, UI/UX 완성도 |
| 약점 | 어르신 건강 신호 분석 없음, 가족 공유 기능 없음 |
| SilverLink 차별점 | 클로바 노트는 "무엇을 말했나"를 기록, SilverLink는 "건강 신호가 있나"를 분석 |

### 2. Google Recorder

| 항목 | 내용 |
|------|------|
| 주 목적 | 개인 메모·오프라인 전사 |
| 강점 | 오프라인 동작, 무료 |
| 약점 | 한국어 지원 제한, 건강 분석 없음, 안드로이드 전용 |
| SilverLink 차별점 | Google Recorder는 기록 도구, SilverLink는 돌봄 도구 |

### 3. 일반 통화 녹음 앱 (ACR Phone, 기본 통화 녹음 등)

| 항목 | 내용 |
|------|------|
| 주 목적 | 통화 증거 보관·법적 분쟁 |
| 강점 | 백그라운드 자동 녹음 |
| 약점 | 분석 기능 없음, 단순 파일 저장 수준 |
| SilverLink 차별점 | 녹음 이후 AI 분석·알림·대시보드로 이어지는 완전한 돌봄 플로우 |

### 4. 카카오 AI 스피커 / 네이버 클로바 케어콜

| 항목 | 내용 |
|------|------|
| 주 목적 | AI가 어르신에게 전화 걸기 (로보콜) |
| 강점 | 어르신이 전화를 받기만 하면 됨 |
| 약점 | 어르신 거부감 높음, 자연스러운 대화 불가, 비용 높음 |
| SilverLink 차별점 | 인위적인 AI 전화 → 이미 하는 가족 통화 강화. "배달의 민족 패턴": 새 행동 추가 아님 |

**가설**: "가족이 이미 주 2~3회 전화하고 있다면, SilverLink는 그 행동에 투명하게 결합된다."

**검증 방법**:
